# Constantinescu 2021 reproducibility

This directory collects the notebook workflow and generated figure assets for
the IMEX-MRK advection-diffusion paper example described in:

- Emil M. Constantinescu, *Implicit extensions of an explicit multirate
  Runge-Kutta scheme*, Applied Mathematics Letters 128 (2022), 107871.

The main notebooks are:

- `TestMRIMEXAdvectionDiffusionPaper.ipynb`
- `TestMRIMEXAdvectionDiffusionPaperPlots.ipynb`

The PNG, EPS, and GIF files next to those notebooks are the generated outputs
used for the paper-oriented reproduction workflow.
